Methodology & Terminology

  • All timestamps converted to US/Pacific (America/Los_Angeles)
  • Fix detection: regex matching fix/fixed/fixes/resolved/address/revert variants
  • Fix-only release: all changelog bullets match fix pattern
  • Changelog entry = one bullet point (“- ” prefix)
  • Version series: major.minor.x grouping (e.g. 2.1.x)
  • Week boundaries: Monday–Sunday
  • Trend window: last 28 days vs. overall
